# High_tension_programmer_Interpreter
"ハイテンション！プログラマー"というネタ言語作成をしたい！

## 仕様
この言語は、とてもフラットな日本語を使用してプログラミングすることができます。
命名に日本語を使用することもできますし、そもそも予約語が日本語です。

## デモコード

### HelloWorld
``` High_tension_programmer
文書保存 helloworld HelloWorld
つぶやく！ helloworld
```
出力：「HelloWorld」

## 予約語解説
文書保存 [変数名] [変数の値]  
→ 文字列を格納する変数を宣言します。  

つぶやく！ [変数名]  
→ 文字列をコンソール画面に出力します。
